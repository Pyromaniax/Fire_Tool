A tool for all of your incendiary needs!<br><br>This tool will help you out with building up fiery scenes and set objects ablaze without having to rely on opening and closing the context menu, or you can use it as a makeshift flamer and lay hell somewhere. It also includes some customizable options as seen in the pictures.<br><br>This addon is a combination of the old [Garry's Mod 12 Ignite Tool](https://maurits.tv/data/garrysmod/wiki/wiki.garrysmod.com/indexb173.html) and [Dicko's IgnitePRO Tool](https://gmods.org/view/7513) with some additional tweaks. Other than those tweaks credits go to Garry and Dicko.<br><br>Primary fire will spawn a customizable [env_fire](https://developer.valvesoftware.com/wiki/Env_fire).<br><br>Secondary will ignite targeted object with customizable length.<br><br>Reload will extinguish the targeted object.<br><br>Yeah, it's just a fancy flamethrower where you have more control over the consequences.<br><br>[Dynamic Fire Glow](https://steamcommunity.com/sharedfiles/filedetails/?id=785842418) and [MMod Particles - Explosion & Fire](https://steamcommunity.com/sharedfiles/filedetails/?id=3102022800") both work with this addon.<br><br>[vFire](https://steamcommunity.com/sharedfiles/filedetails/?id=1525218777)  is only partially supported as it doesn't replace env_fire, only the ignition will be replaced.</div><br>Have fun little pyros!<br><br> Alternate file sources:<br><br>[Steam Workshop](https://steamcommunity.com/sharedfiles/filedetails/?id=2189698828)<br><br>[GameBanana](https://gamebanana.com/mods/528753)
